# 중국어 국내 문헌 사각지대 — 수집 경로 설계 (Step 3)

## 왜 필요한가
OpenAlex 코퍼스는 사실상 **영어권 위주**다(파일럿 표본 1,237편 중 언어 `en` 99%).
특허가 "95%가 국내(중국어) 출원"인 것과 동형으로, 중국 연구 상당수는 **중국어·국내 학회/학술지**에
발표돼 영어권 색인에 안 잡힌다. 이걸 빼면 중국 생태계를 절반만 본다.

## 후보 소스 (실현 가능성 순)

| 소스 | 커버리지 | 접근성 | 비고 |
|------|----------|--------|------|
| **CNKI (中国知网)** | 중국 최대 학술 DB, 학위논문·회의록 포함 | ⚠️ 유료·기관구독·크롤 제약 강함 | 가장 포괄적이나 대량 수집 어려움. 기관 구독 API 또는 수동 표본 |
| **万方 (Wanfang)** | 학술지·학위·회의 | ⚠️ 유료·API 제한 | CNKI 보완재 |
| **维普 (VIP/CQVIP)** | 중문 학술지 | ⚠️ 유료 | 3대 중문 DB 중 하나 |
| **CSCD (중국과학인용DB)** | 핵심 중문 저널 인용 | ⚠️ 제한 | 인용분석용 |
| **OpenAlex 중국어 부분** | `language='zh'` 필터 | ✅ 이미 보유 | 소량이나마 무료·즉시. 1차 확인용 |
| **arXiv (중국 저자)** | 영문 프리프린트 | ✅ 무료 | 언어는 영어지만 국내 미공개 연구 선점 포착 |
| **특허 NPL(비특허 인용)** | 특허가 인용한 중국어 논문 | ✅ 이미 보유(EPO) | 연구→특허 연결의 중국어 논문 목록 확보 가능 |

## 권장 접근 (단계적, 비용·법적 리스크 최소화)

1. **무료 1차**: OpenAlex `language='zh'` 부분집합 추출 + 특허 NPL의 중국어 참조 목록 정리
   → "얼마나 큰 사각지대인가"를 무료로 정량화(영어 대비 중국어 비중 추정).
2. **표본 확보(수백 편)**: CNKI/万方에서 **주제 검색 → 서지+초록 표본** 수집.
   - 기관(대학·KISTEP 등) 구독 계정으로 합법적 다운로드, 또는 공식 API 협의.
   - 전수(수만 편)가 아니라 **대표 표본**으로 충분(정성 패턴 파악이 목적).
3. **번역·추출 통일**: 중국어 초록 → 동일 `schema.json`으로 LLM 추출(Claude는 중국어 직접 처리 가능,
   별도 번역 불필요). 영어 코퍼스와 **같은 계기로 측정**해야 CN(중문) vs CN(영문) vs KR vs US 비교 성립.

## 분석 설계에의 함의
- 결과 보고 시 **3개 층위**를 구분: ① CN 영어논문(OpenAlex) ② CN 중국어논문(표본) ③ KR/US.
- 가설: 중국어 논문은 **응용·시스템통합·국내지향 비중이 더 높고 FWCI 낮음**
  (특허의 국내집중과 동형) → 검증 대상.
- 한계 명시: 중국어 표본은 전수가 아님(선택편향 가능), CNKI 접근 제약.

## 즉시 실행 가능한 무료 작업 (다음 스텝 후보)
- `build_pilot_corpus.py`의 언어 필터를 `zh` 포함으로 바꿔 중국어 부분집합 규모 측정.
- 특허 전수 레코드(`data/patents/kr_cn_full.jsonl`)의 NPL 필드에서 중국어 논문 참조 추출.
